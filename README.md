# Student Marks Predictor

A web application that predicts student marks based on input features using Python and Flask.

## Features
- Predict marks using a trained ML model
- Simple web interface with input forms
- Display predicted marks instantly

## Technologies Used
- Python
- Flask
- HTML/CSS
- Machine Learning (scikit-learn)

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python web_app/app.py`
4. Open browser at `http://localhost:5000`
